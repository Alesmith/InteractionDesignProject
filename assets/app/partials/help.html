Hjälp

Webbportalen orange. fungerar som ett verktyg för studenter på Göteborgs universitet och Chalmers att se sina skrivna tentamina digitalt. Om du har svårigheter med att hitta det du letar efter på webbplatsen kan du läsa beskrivningen nedan. I menyn till höger kan du navigera dig snabbare för att hitta ett svar på din fråga.

Hjälp-sidan är enbart till för att stödja dig i navigeringen på webbplatsen, och inte för att ge information rörande tentaminaprocessen. För information om tentamina ber vi dig se fliken för Information.

Om du inte skulle hitta ett svar på din fråga kan du alltid ta kontakt med studiexpeditionen, orange. eller Alesmith. Deras kontaktuppgifter hittar du under fliken för Information.

Ditt konto

  • Ändra lösenord

    För att ändra ditt lösenord klickar du på menyalternativet Inställningar.

  • Ändra telefonnummer

    För att ändra ditt telefonnummer klickar du på menyalternativet Inställningar.

  • Ändra aviseringsmetod

    För att ändra den aviseringsmetod som ger dig information om uppdateringar på .orange klickar du på menyalternativet Inställningar. Där kan välja mellan att få aviseringar via mail, sms eller ta bort aviseringarna helt och hållet.

  • Ändra dina personuppgifter

    Dina personuppgifter, vilka inkluderar personnummer, förnamn och efternamn hämtar orange. direkt från GUL. Du har därför inte möjlighet att ändra dessa uppgifter direkt på webbplatsen. Om din personuppgifter inte skulle stämma ber vi dig att kontakta Studieexpeditionen så hjälper de dig att uppdatera dem. Kontaktinformation till studieexpeditionen hittar du under alternativet Information.

  • Ändra din mailadress

    Din mailadress hämtas direkt från GUL och är något du får som student på Göteborgs universitet eller Chalmers. Du har därför inte möjlighet att ändra din mailadress.

  • Ändra dina profilbild

    För att ändra din profilbild går du in på menyalternativet Inställningar.

  • Vid problem med ditt konto

    Om ditt konto inte fungerar som det ska ber vi dig kontakta support hos orange. Kontaktuppgifter till orange. hittar du under menyalternativet Information.

Tentamen

  • Se dina tentor

    På startsidan och under menyalternativet Tentamen kan du se en överblick av alla dina skrivna tentamina. Där finns information om respektive tenta i form av kursnamn, kurskod, tentamendatum och högskolepoäng för tentamen.



    Om din tentamen registrerats på .orange kommer du ha möjlighet att se ditt betyg och även klicka på knappen "Visa tentamen" för att se hela din digitala tentamen. Om din tentamen däremot inte registrerats visas detta genom att du inte har möjlighet att klicka dig vidare för att se den digitala tentan. Det ska då även finns en nedräkning för när tentamen beräknas bli registrerad.

  • Se dina betyg

    Betyget för din tentamen visas direkt i överblicken av dina tentamina på startsidan, alternativt under menyalternativet Tentamen.

  • Se dina poäng

    De totala poängen för din tentamen och för respektive uppgift kan du se när en tentamen registrerats på orange. När du valt att "Visa tentamen" på startsidan (alternativt under menyalternativet Tentamen) vid din registrerade tentamen visas dina poäng.

  • Se din feedback

    Om examinatorn har valt att ge dig feedback på tentamen har du möjligt att se det när du valt att "Visa tentamen" på startsidan (alternativt under menyalternativet Tentamen). Din text kommer då markeras på de områden där examinatorn har gjort kommentarer. När du håller muspekaren över dessa markeringar kommer kommentaren visas.

  • Se vem som är tentamenansvarig

    Vilken person som är ansvarig för din tentamen visas om din tentamen registrerats på orange. När du valt att "Visa tentamen" på startsidan (alternativt under menyalternativet Tentamen) vid din registrerade tentamen visas all information rörande den valda tentamen.

  • Spara eller skriv ut din tentamen

    När du valt att "Visa tentamen" på startsidan (alternativt under menyalternativet Tentamen) vid din registrerade tentamen har du möjlighet att skriva ut och/eller spara den.

Meddelanden

  • Se dina meddelanden

    Dina meddelanden kan du hitta under menyalternativet Meddelanden. Där finns alla konversationer för varje tentamen med respektive examinator.

  • Ta kontakt med en tentamenansvarig

    För att ta kontakt med en tentamenansvarig när du vill diskutera din tentamen måste du gå in på den tentamen du vill diskutera genom att välja "Visa tentamen" på startsidan (alternativt under menyalternativet Tentamen). Där kan du välja att "Kontakta examinator". En ny konversation kommer då skapas mellan dig och din examinator vilket gör det möjligt för dig att diskutera ditt resultat. Du kan även gå in på Meddelanden och välja den konversation som handlar om den tentamen du vill diskutera
